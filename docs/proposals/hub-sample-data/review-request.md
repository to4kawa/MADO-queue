# Review request: MADO hub sample data proposal

Please review whether the draft table granularity and columns are useful for discussion across MADO hub, MADO form, MADO care, and MADO move workflows.

CSV files have been added as the primary sample-data review format so maintainers can more easily inspect the proposed columns and fictional values. `schema.sql` defines the draft table structure, and `seed.sql` is retained only as secondary/reference material.

This review request is not asking maintainers to approve a production schema. It is also not claiming compliance with the resident record system standard specification. Public specifications are used only as references for discussion.

All records are fictional. The sample does not include real municipality, resident, address, or personal data.
